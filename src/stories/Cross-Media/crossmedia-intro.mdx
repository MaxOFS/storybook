# Accessibility
To ensure consistency across all publications, all visualisations should have the same anatomy (order of elements). Some of the elements are optional. These can be omitted accordingly. However, the order and spacing of the elements should always be kept the same.

On the following pages you will find the dimensions (spacing of the individual elements), as well as the font styles for [desktop](?path=/docs/cross-media-layout-desktop--docs), [mobile](?path=/docs/cross-media-layout-mobile--docs) and [social media platforms](?path=/docs/cross-media-layout-social-media--docs).

The height of the visualisation is variable, the width is to be adapted to the respective platform on which the graphics are published. Whereby it should be noted here that it is a maximum or minimum width specification in each case. The graphics behave responsive on the web, of course.
