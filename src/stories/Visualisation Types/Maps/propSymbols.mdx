## Description
This map type will display a comparison among values of one continuous category, which may include negative values. The data values are shown proportionally.

## Do and don't
- Scale symbol area proportionally to the value
- Include a graduated size legend
- Use for absolute counts that cannot be normalised
- Anchor symbols to a specific position (typically the centroid of a region)
- Avoid symbols that grow so large they obscure the base map
- Avoid visualisation of relative data (rates, percentages, etc.)
