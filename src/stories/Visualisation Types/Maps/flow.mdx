## Description
Flow maps show movements or flows of data between locations. They can be used to visualise qualitative (categorical) or quantitative (numerical) data. Qualitative data is represented by colours, whilst quantitative data is visualised using line thickness, which illustrates the quantitative relationships. Arrows indicate the direction of the data flow, whilst points can additionnaly indicate the quantity of a second data set.

## Do and don't
- Radial lines can create an optical illusion, which helps to better represent a route. With a small dataset, the curvature also makes it possible to avoid overlaps with points or other flows.
- When dealing with large amounts of data, the visualisation should look as simple as possible. As such, straight-line flows are used, as these are easier to read. 
- In principle, any layout can be used. However when dealing with large amounts of data, it is important to ensure good readability, which is why the data should, where necessary, be reduced or made easier to understand through additional interactivity.
