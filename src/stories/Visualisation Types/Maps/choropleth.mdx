## Description
A Choropleth Quantitative Map shows how an indicator/variable changes across a geographic area. This map type displays the values of one continuous category with a color gradient (sequential color scheme), in relation to a statistical area (administrative unit) or an entity (ex: inhabitant). The data may include negative values, in which case there will be a different color for positive and negative values (qualitative color scheme)
