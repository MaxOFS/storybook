## Description
A Choropleth Quantitative Map shows how an indicator/variable changes across a geographic area. This map type displays the values of one continuous category with a colour gradient (sequential colour scheme), in relation to a statistical area (administrative unit) or an entity (ex: inhabitant). The data may include negative values, in which case there will be a different colour for positive and negative values (qualitative colour scheme).

## Points of attention
- Avoid visualisation of absolute data (integer values)
- Avoid too many categories
- Include a complete legend with class boundaries
- Choose a classification method appropriate to the data
