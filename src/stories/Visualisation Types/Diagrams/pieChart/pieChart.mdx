## Description
A pie chart is a common way of showing part-to-whole data. It is a circle divided into sectors corresponding to the expressions of a qualitative variable. It is used to show the proportion of each sector in relation to the total number. The length of the arc, as well as the angle of each sector, is proportional to the value represented.
It is important to avoid comparing values with a pie chart. It is better to use another type of chart.

## Points of attention
- Be aware that it’s difficult to accurately compare the angle of each part
- Avoid Negative values
- Show the legend with the value inside the chart
- Show total value in legend (if the values are not in %)
- Pay attention to the treatment of missing values
- Avoid too many categories (3 recommended, maximum 5)
- Skinny slices are difficult to read
