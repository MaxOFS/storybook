## Description
A Butterfly Chart allows two variables that share the same modalities to be compared side by side. In this way, the differences between the variables can be quickly identified. A Butterfly Chart presents 2 bar charts back-to-back. The categories or intervals are presented vertically.

## Points of attention
- Categories should be equal in size (intervals of equal size)
- Both X-axes need the same scale
- When there is a Zero-line or Base-line, it should be highlighted in black
- The 2 variables can be distinguished by colour
