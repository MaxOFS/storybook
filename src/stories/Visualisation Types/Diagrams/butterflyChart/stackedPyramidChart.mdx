## Description
A stacked Pyramid chart is similar to a standard pyramid chart in which it illustrates the distribution of various age groups. The main difference being that it showcases the distribution of multiple groups.

## Points of attention
- Do not use if you want to represent precise values, this type is best used to convey trends.
- Use only when you have more than 3 age groups
