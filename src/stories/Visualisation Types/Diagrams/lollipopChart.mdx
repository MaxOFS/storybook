## Description
A Lollipop Chart replaces the bars in a Bar Chart with lines and dotss. This type of chart is preferable when there is a lot of data or when the reprented values are all relatively similar. Lines and dots in these cases are easier to visualize. In fact, Lollipops draw more attention to the data value than standard bars/columns and can also display rank and value effectively.
