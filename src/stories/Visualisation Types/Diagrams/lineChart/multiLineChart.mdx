## Description
A multiple line chart provides a clear way to compare the evolution of several numerical variables over time. 
It is made of various variables respectively represented by points which are connected by a line. It generally displays changes over a period of time but is also used to visualize distributions with a large amount of data (ex: normal distribution by Gauss).

The X-axis usually represents a continuous (often time) quantitative variable. The Y-axis represents a numerical value.

## Do and don't
- The last value is always highlighted with an outlined circle and the value in bold and black.
- On Desktop Labels are directly placed next to the value, (if they are not too long (no more than 2 lines))
- Y-axis: should generally start at zero
- X-axis: time intervals should be equal
- The number of lines per graph may never exceed the number of 4, but it depends primarily on the dynamics of the data set.
- The more the lines cross, the fewer variables can be represented.
