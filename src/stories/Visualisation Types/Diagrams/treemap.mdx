## Description
A treemap or tile chart represents hierarchical data in the form of nested rectangles, which in turn may contain other rectangles. It is a way to display the proportion of each modality as a function of the total, as an alternative to the pie chart.

## Points of attention
- If the data has too many small categories, the visualisation will be difficult
- The data has to be hierarchical
- Exact comparison between the categories should not be a primary concern
- If optional colour variable is used, a Legend will be necessary
