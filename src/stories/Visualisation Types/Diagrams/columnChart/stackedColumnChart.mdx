## Description
A Stacked Column Chart is a simple way of showing part-to-whole relationships but can be difficult to read with more than a few components. The size of the components is more effective to read in a Stacked Column Chart than in a pie chart. 

A Stacked Column Chart uses color differences to show several dimensions. To do this, each bar is split into a set of dimension items, distinguished by color, while maintaining the total bar height.

A vertical layout is the best choice for short labels and trends.
