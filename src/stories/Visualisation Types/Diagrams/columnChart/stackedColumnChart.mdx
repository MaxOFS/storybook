## Description
A Stacked Column Chart is a simple way of showing part-to-whole relationships but can be difficult to read with more than a few components. The size of the components is more effective to read in a Stacked Column Chart than in a pie chart. 

A Stacked Column Chart uses colour differences to show several dimensions. To do this, each bar is split into a set of dimension items, distinguished by colour, while maintaining the total bar height.

A vertical layout is the best choice for short labels and trends.

## Do's and don'ts
- In the case of a single set of categories on the X-axis, it is best to order them in function of Y values (from biggest to lowest) for a better visualisation output
- Exception made of the first colour item (closest to the X-axis), the values of the coloured items represented are added on top of the previous category, this means that the exact value can’t be read on the graph but needs to be calculated. What is shown is the relative importance of each group and the total of the stacked variables.
- Time intervals should be equal
- Avoid too complex data
- Avoid too many colour categories
