## Description
A bump chart is used to showcase the changes in rank over time of multiple groups or individuals. This type of chart is effective in showing trends, comparing evolutions. A notable example of a bump chart is the one made by GitHub to showcase [the top programming languages](https://github.blog/news-insights/octoverse/octoverse-a-new-developer-joins-github-every-second-as-ai-leads-typescript-to-1/)
