## Description
In a sankey or alluvial diagram variables are assigned to vertical axes that are parallel. Values are represented with blocks on each axis. The height of a block represents the size of the cluster and the height of a stream field represents the size of the components contained in both blocks connected by the stream field.

## Do and don't
- The nodes should be arranged in an appropriate manner
- There shouldn’t be too many flows with small quantities
- Ther shouldn’t be a single flow dominating
- There shouldn’t be too many individual flows
- For better visualisation, all the Y-axes should be ordered from biggest to smallest category block
- There can’t be negative values
