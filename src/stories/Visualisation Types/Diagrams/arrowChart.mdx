## Description
The arrow chart is a specialised form of the connected dot plot (dumbbell plot). While a dumbbell plot treats both endpoints symmetrically, the arrow chart introduces directionality: an arrowhead at the end point unambiguously signals whether a value has increased or decreased, and by how much. This makes it particularly effective for before/after comparisons, temporal evolution, or any transition between two states.

Categories are displayed along the vertical axis; the horizontal axis carries the shared quantitative scale. Each row is one category, represented by a single arrow spanning from the start value to the end value. Colour and arrowhead direction encode whether the change is positive or negative.
