## Description
A stacked Area Chart shows the evolution of the variable along the X-axis, but also its relationship with the other variables and with the total (along the Y-axis). Each variable is represented by an individual, colored area.
The variables are stacked upon each other. In other words, each variable is stacked upon the previous in such a way that the previous serves as base-line to the next. The area between the X-axis and the top variable represents the total of all the variables.

This allows to check on the same figure the evolution of both the total of a numeric variable, and the importance of each group. 

Use the stacked area chart to show trends over time among related items.

## Do's and don'ts
- Y-axis should start at zero
- Exception made of the first line (closest to the X-axis), the values represented are added on top of the previous category, this means that the exact value can’t be read on the graph but needs to be calculated. What is shown is the relative importance of each group and the total of the stacked variables.
- Negative values can’t be treated
- Time intervals should be equal
