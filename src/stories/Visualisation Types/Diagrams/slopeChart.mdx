## Description
A slope chart is a line chart that represents a variable at two different points in time.
The time is shown horizontally, while the values are placed vertically. The evolution is represented by the slope.
A parallel coordinate chart is an extension of the slope chart with more than two different time points.

## Do's and don'ts
- Labels are – if they are not too long – directly placed next to the value.
- The last value is always highlighted in bold and black, if a label is placed next to it.
- Avoid too complex data
- All Y-axes should have the same scale
