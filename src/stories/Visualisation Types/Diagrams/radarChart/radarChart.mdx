## Description
A radar chart (also known as spider charts, polar charts, web charts, or star plots) presents a way to visualise multivariate data. It is used to plot one group of values over various shared variables. 
Each shared quantitative variable is represented by an axis starting at the center. All axes are radial. equally spaced, and of the same scale. The data is diusplayed on each axis and the connection of all data points forms a polygon.

## Do's and don'ts
