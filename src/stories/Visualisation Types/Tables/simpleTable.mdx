## Description
A simple table represents data in the form of a grid with lines and columns. It is used to show structured and readable information.
Lines are used to represent observations, organized in a hierarchical manner. Columns are assigned a variable each. https://data.europa.eu/apps/data-
visualisation-guide/intro-to-tidy-data
