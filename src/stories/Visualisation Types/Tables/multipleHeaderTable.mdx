## Description
A multiple header table is similar to a hierarchical table but the groups and hierarchical levels are based on the variables (columns) and not the observations (lines) of the grid.
Multiple header tables are not 100% machinereadable. Because merged header cells or multilevel headings break the predictable row/column associations required by machines.

## Points of attention
- Not too many columns
- One subject per table
- No cell fusion to guarantee machine readability
