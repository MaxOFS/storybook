## Description
A table with a visualisation is a hybrid table containing numerical data and graphical elements in the same grid. It is used in order to put emphasis on certain trends, comparisons or rankings by integrating mini-graphs (bars, sparklines or icons) in the cells of a table.

Lines are used to represent observations, organized in a hierarchical manner. Columns are assigned a numerical variable or visual elements, if they add something to the readability, each.
