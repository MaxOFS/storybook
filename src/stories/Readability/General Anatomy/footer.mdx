# Footer

The footer is the institutional baseline of a chart. It carries fixed administrative metadata : the copyright notice, the FSO publication number (BFS-Nummer), the data reference date, and the source. Unlike the notes area, which is contextual and varies by chart, the footer follows a standardised format applied consistently across all FSO outputs.

The footer is not optional in official FSO publications. Its elements are defined by institutional conventions and must not be rearranged, abbreviated, or omitted.

## Description

The footer contains four elements, always presented in the same order and layout:

- **Data reference date** : the date or period to which the data refers, expressed in a standardised format. Distinct from the publication date: a chart published in 2024 may refer to data from 2023 or earlier.

- **Copyright** : the standard FSO copyright notice, asserting institutional ownership of the chart and its underlying data representation. Follows the form *© FSO [year]* or the full federal formulation as specified by the CD Bund guidelines.

- **Source** : the data source, following the same citation format as in the notes area (*Source: FSO, [dataset], [year]*). When the chart's notes already carry a source line, the footer source must be consistent with it; the footer version may be more condensed if space requires.

- **BFS-Nummer** : the FSO internal publication reference number that uniquely identifies the statistical product or publication the chart belongs to. Enables traceability back to the official publication catalogue.


These four elements are typically set in two compact lines at the very bottom of the figure, visually separated by a padding from the notes area above.
