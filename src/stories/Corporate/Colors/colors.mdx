# Introduction to Colors
In order to create a clear recognition of the FSO graphics, the color palette was completely revised. In addition, it was ensured that all colors are barrier-free and comply with the [Web Content Accessibility Guidelines (WCAG) 2.1](https://www.w3.org/TR/WCAG21/).

The colors of a graphic are primarily intended to help the reader distinguish (dissociate) or connect (associate) things. However, the graphic creator must always try to use colors in a way that supports the message and readability. The same applies to the legend - it should be as logical and easy to read as possible.

The color should be used with care and as a complement to other visual variables such as position, shape or size. It is therefore recommended to use colors in the FSO's visualisations with moderation in order to maintain neutrality and objectivity. 

In order to make the application of colors easier, the new color palettes are divided into the following groups :

## [Themes colors](?path=/docs/corporate-colors-themes-colors--docs)
To ensure consistency across charts that explore recurrent themes, the colors have been carefully assigned to represent the different themes in a meaningful way.
The following themes have been developed, which should always be represented with the according color palette: Languages, Religions, Economic Sectors, Education, Voting, Gender.

## [Political parties colors](?path=/docs/corporate-colors-political-parties-colors--docs)
A separate color palette has been developed for the political parties in order to clearly distinguish them from each other. It is mandatory that this color palette is “always” and “only” used in the context of political content. This color palette may not be used for other content.
