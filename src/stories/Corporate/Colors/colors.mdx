# Introduction to Colours
In order to create a clear recognition of the FSO graphics, the colour palette was completely revised. In addition, it was ensured that all colours are barrier-free and comply with the [Web Content Accessibility Guidelines (WCAG) 2.1](https://www.w3.org/TR/WCAG21/).

The colours of a graphic are primarily intended to help the reader distinguish (dissociate) or connect (associate) things. However, the graphic creator must always try to use colours in a way that supports the message and readability. The same applies to the legend - it should be as logical and easy to read as possible.

The colour should be used with care and as a complement to other visual variables such as position, shape or size. It is therefore recommended to use colours in the FSO's visualisations with moderation in order to maintain neutrality and objectivity.

## [Themes colours](?path=/docs/corporate-colours-themes-colours--docs)
To ensure consistency across charts that explore recurrent themes, the colours have been carefully assigned to represent the different themes in a meaningful way.
The following themes have been developed, which should always be represented with the according colour palette: Languages, Religions, Economic Sectors, Education, Voting, Gender.

## [Political parties colours](?path=/docs/corporate-colours-political-parties-colours--docs)
A separate colour palette has been developed for the political parties in order to clearly distinguish them from each other. It is mandatory that this colour palette is “always” and “only” used in the context of political content. This colour palette may not be used for other content.
