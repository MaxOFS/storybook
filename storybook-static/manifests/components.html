# Introduction to Colors
In order to create a clear recognition of the FSO graphics, the color palette was completely revised. In addition, it was ensured that all colors are barrier-free and comply with the [Web Content Accessibility Guidelines (WCAG) 2.1](https://www.w3.org/TR/WCAG21/).

The colors of a graphic are primarily intended to help the reader distinguish (dissociate) or connect (associate) things. However, the graphic creator must always try to use colors in a way that supports the message and readability. The same applies to the legend - it should be as logical and easy to read as possible.

The color should be used with care and as a complement to other visual variables such as position, shape or size. It is therefore recommended to use colors in the FSO's visualisations with moderation in order to maintain neutrality and objectivity.

## Description
An area chart is used to visualize changes to a metric value over time. Because it has only one metric, it serves the same function as a simple line chart. An area chart is distinguished from a line chart by the addition of shading between lines and a baseline, like in a bar chart.

### Area chart with diverging areas
A Diverging Area Chart will present a standard area chart but with a diverging display of the data around a Base-line or Zero-line. It is used to compare values and to highlight 2 directions in the data over a single dimension. It should be used when the data includes negative and positive values. ## Description
A stacked Area Chart shows the evolution of the variable along the X-axis, but also its relationship with the other variables and with the total (along the Y-axis). Each variable is represented by an individual, colored area.
The variables are stacked upon each other. In other words, each variable is stacked upon the previous in such a way that the previous serves as base-line to the next. The area between the X-axis and the top variable represents the total of all the variables.

This allows to check on the same figure the evolution of both the total of a numeric variable, and the importance of each group. 

Use the stacked area chart to show trends over time among related items.

## Description
The arrow chart is a specialised form of the connected dot plot (dumbbell plot). While a dumbbell plot treats both endpoints symmetrically, the arrow chart introduces directionality: an arrowhead at the end point unambiguously signals whether a value has increased or decreased, and by how much. This makes it particularly effective for before/after comparisons, temporal evolution, or any transition between two states.

## Description
Boxplot charts are used to summarize multiple distributions of quantitative data. It displays a central rectangle which extends from the 25th quantile (Q25) to the 75th quantile (Q75) and which is intersected by a line representing the median (Q50 or 50th quantile). This median line displays the value for which half of the data is below and the other half is above. The lines outside the rectangle are called whiskers and generally represent the 10th and 90th quantile (Q10 and Q90). Outliers may be plotted as individual points. The spacings between the different parts of the box indicate the degree of dispersion (spread) and skewness (asymmetry) in the data, and show outliers.

## Description
A radar chart (also known as spider charts, polar charts, web charts, or star plots) presents a way to visualise multivariate data. It is used to plot one group of values over various shared variables. 
Each shared quantitative variable is represented by an axis starting at the center. All axes are radial. equally spaced, and of the same scale. The data is diusplayed on each axis and the connection of all data points forms a polygon.

# Bubble Chart
## Table of contents
- [Description](#description)
- [Anatomy](#anatomy)
- [Chart variations](#chart-variations)
- [Do's and don'ts](#dos-and-donts)

## Description
A Bubble Chart is used to look at the relationships between three numerical variables. The first and second variable will be plotted in function of values along the X and Y-axes as a scatterplot. The third variable will determine the size of the points, and therefore their respective weight.
It is important that the value of the third variable corresponds to the area of the bubble and not its diameter. In order to avoid hiding the small circles with the overlaying of big circles, it is mandatory to plot the smallest circles in front of the larger ones.

## Chart variations

## Do's and don'ts
- Superimposition treatment needs attention, small points in front of big points, to avoid loss of information
- The size/weight variable should be represented by the area of point and not the diameter
- The radius of the symbols is not the square root of the amount. Flannery correction should be applied
- Axes should start at zero
- Labels are mandatory
- Optional labels should not be superimposed
- Avoid too many superimposed elements
